# CRUD #

## Objetivo ##

Praticar as operações do CRUD em uma aplicação simples. Não trata de questões de segurança e nem relacionamento entre tabelas.

A aplicação visa registrar uma espécie de conta de luz. Recebendo do usuário os dados pessoais e de consumo de energia.

### Tecnologias usadas ###

- HTML
- CSS
- Bootstrap
- PHP
- MySQL
